# CRM System

## Project Overview

This project is a CRM (Customer Relationship Management) system developed using **Core PHP** and **MySQL**. It is designed to manage end-to-end Sales, Billing, and HR operations in a single application.

## Features

* User Authentication
* Customer Management
* Product Management
* Sales Management
* Billing & Invoice Generation
* HR Management
* Reports & Dashboard

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL

## Project Status

🚧 Currently in the planning and database design phase.
